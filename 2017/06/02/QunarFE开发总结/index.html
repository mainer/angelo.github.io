在手机上实时查看代码改动

使用手机浏览器 Debug ，要让手机浏览器访问的网络定向到开发机的服务，因此需要设置代理。

开发机、手机需要在同一局域网。

开发机上启动本地服务器，该服务器的地址是：本机当前网络的 IP 地址，端口号为 999 :

手机在 Wifi 处设置代理，代理 IP 为开发机当前网络的 IP，端口号为 999，设置后，手机中的所有请求都会被代理到该 IP 和端口所指的服务程序，从而访问到 Debug 页面。

如果你要在 PC 浏览器上访问开发页面，PC 浏览器也需要设置代理地址和端口，从而将浏览器的请求代理到本机服务器程序。代理地址为：localhost:999。

NOTE：浏览器使用SwitchOmega Chrome 插件来管理代理地址。

使用 IconFont

  1. 去 ICON-FONT-ADDRESS 我的图标项目中查看是否有自己引入的图标，如果有，即可使用；如果没有，则进入第二步：
  2. 去 ICONS-库 找到自己需要新加的图标，保存到购物车，保存到自己的项目。
  3. 进入步骤 1 我的项目页面，点击 同步Source 按钮，生成新的图标库版本。此时线上就会使用最新的图标库版本。
  4. 进入工程：busapp2/src/yo-config/core/iconfont.scss 文件中。修改对应的iconFont 库版本即可。
  5. 重新 ykit pack -m 打包工程，刷新页面即可看到新的图标已经可以使用。
