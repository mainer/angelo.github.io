自动部署脚本

创建名为 hd 的文件， 修改其权限 chmod 755 hd, 将其放到 /usr/local/bin 目录下。这样就可以在终端的任意位置使用 hd 命令一键发布最新写的博客。

#!/bin/sh
cd ~/an/blog
hexo clean
hexo g
hexo d
