Qunar FE开发总结

发表于 2016-10-11 | 分类于 Tech

一些链接：

  1. 核心组件-Yo
    Yo 是一个专注于移动开发的 Front-end UI Framework，她轻量，易用，可配置，并且具备超强的扩展能力。

  2. Ykit工具
    Ykit 是一套可配置和可扩展的前端开发工具集，让开发者灵活地搭建开发环境，和使用资源打包、静态资源访问、网络代理等服务。

  3. Qunar UED
    合集

环境配置

  1. 安装 qnpm ykit 等插件

  2. Clone Project busapp2

  3. 配置hosts文件：

    ADD NEW LINE： 127.0.0.1 q.qunarzz.com
    NOTE:
    (1) 上面的地址会影响到 发版万事屋(ADDRESS) 的访问
    (2) 推荐使用SwitchHosts以便捷切换hosts，SwitchHosts官方主页

  4. 执行qnpm install以安装所有必须的插件

  5. 打包工程，在工程目录下执行：ykit pack -m
  6. 在工程的上一目录启动服务：sudo ykit server
  7. 在Chrome浏览器输入以下链接，并尝试修改页面查看结果：
    http://hy.bus.qunar.com/coachHy/#/charteredDemandPage

在手机上实时查看代码改动

  1. 手机-WifiA-代理A地址：手机浏览器的请求都会发送到A地址指示的服务器；
    手机直接在Wifi处设置代理地址。

    PC浏览器-WifiA-浏览器代理A地址：PC浏览器的请求会发送到A地址指示的服务器；
    浏览器使用SwitchOmega Chrome插件来管理代理地址；

  2. sudo ykit server -x:
    启动本地代码服务器，该服务器的地址是：本机当前网络的IP地址，端口号为999

  3. 本地修改代码，保存，会自动编译，并上传到本地服务器。手机和PC浏览器就可以访问到了。

iconFont使用

  1. 去ICON-FONT-ADDRESS 我的图标项目中查看是否有自己引入的图标，如果有，即可使用；如果没有，则进入第二步：
  2. 去ICONS-库找到自己需要新加的图标，保存到购物车，保存到自己的项目。
  3. 进入步骤1我的项目页面，点击同步source按钮，生成新的图标库版本。此时线上就会使用最新的图标库版本。
  4. 进入工程：busapp2/src/yo-config/core/iconfont.scss文件中。修改对用的iconFont库版本即可。
  5. 重新ykit pack -m打包工程，刷新页面即可看到新的图标已经可以使用。
